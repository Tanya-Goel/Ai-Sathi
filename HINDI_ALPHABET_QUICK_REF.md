## Exercise Types Per Lesson

Each alphabet lesson includes:
1. **Vocab Viewer**: Learn letters with audio pronunciation
2. **Matching Game**: Match letters with romanization

Each reading lesson includes:
1. **Vocab Viewer**: Learn words with meanings
2. **Matching Game**: Match words with English meanings

## Progression Logic

- тЬЕ **Lesson 1** is unlocked by default
- ЁЯФТ **Lessons 2-17** unlock sequentially upon completion
- ЁЯУК Students must complete alphabet (1-14) before vocabulary (15-17)
- ЁЯОп Each lesson must be completed to unlock the next

## Key Learning Outcomes

By completing this curriculum, students will:

1. тЬЕ Recognize all 12 Hindi vowels
2. тЬЕ Recognize all 36 Hindi consonants
3. тЬЕ Understand the varga (group) system
4. тЬЕ Read and write matras (vowel signs)
5. тЬЕ Read simple 2-3 letter Hindi words
6. тЬЕ Read words with matras
7. тЬЕ Build basic Hindi vocabulary
8. тЬЕ Have foundation for reading Hindi texts
